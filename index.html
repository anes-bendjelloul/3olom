تطبيقات تحليل نحوي

تطبيقات تفاعلية لتحديد العامل والمعمول، تحليل الإعراب، وتصحيح الأخطاء النحوية بناءً على المحاضرات.

⚖️

تطبيقات الوزن الصرفي

تدريبات على تحديد الوزن الصرفي، استخراج الجذر، وتحليل البنية الصرفية للكلمة العربية.

📋

نظام اختبارات متقدم

اختيار متعدد، صح/خطأ، ملء الفراغ، تحليل نحوي وصرفي. تصحيح تلقائي وعرض النتيجة فوراً.

متابعة التقدم الدراسي

🎬

محاضرات محمية

فيديوهات الشرح، الملخصات، مخططات الدروس، والمواد التعليمية متاحة للمشاهدة فقط.

👨‍💼

لوحة تحكم الأستاذ

📱

تطبيق هاتفي متكامل

📜 ميثاق النزاهة العلمية

كل طالب يلتحق بالمنصة يُوقِّع على ميثاق النزاهة العلمية قبل تفعيل حسابه، تأكيداً لمبادئ البحث الأكاديمي الرصين والتعامل الأخلاقي مع أدوات الذكاء الاصطناعي.

  • ✦ الالتزام التام بالأمانة العلمية في جميع التقييمات
  • ✦ استخدام الذكاء الاصطناعي أداةً مساعدة لا للغش أو الانتحال
  • ✦ احترام حقوق الملكية الفكرية للمحاضرات والمواد العلمية
  • ✦ الإسهام البنّاء في النقاشات الأكاديمية بأسلوب علمي رصين
  • ✦ قبول إشراف الأستاذ ومتابعته الأكاديمية المستمرة

💾 الطريقة 1: التصدير والاستيراد

هذه الطريقة تعمل فوراً ولا تحتاج لإعدادات معقدة. استخدمها لمشاركة البيانات مع زملاء آخرين.

🔹 الخطوات:
1. اضغط "💾 تصدير البيانات" لتنزيل ملف بيانات الطلبة
2. انقل الملف إلى الأستاذ أو الشخص الآخر
3. هو يضغط "📥 استيراد البيانات" ويختار الملف
4. البيانات ستُحدث عنده فوراً ✅

⚠️ ملاحظة: عند تصدير، يتم حفظ جميع بيانات الطلبة في ملف JSON يمكن مشاركته بسهولة.

🔄 الطريقة 2: مزامنة فورية (Firebase)

هذه الطريقة توفر مزامنة فورية بين كل المستخدمين بدون تدخل يدوي (أفضل للفريق الدائم).

🔹 الإعداد:
1. اذهب إلى Firebase Console
2. أنشئ مشروع جديد
3. أضف Realtime Database
4. انسخ بيانات المشروع (API Key, Project ID, إلخ)
5. اتصل بالمطور لتحديث الإعدادات في الكود

💡 Firebase مجاني ✓ | آمن ✓ | سريع جداً ✓
